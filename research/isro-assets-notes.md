# ISRO Assets Notes

## Purpose

This file will track candidate Indian satellite assets for the Protect ISRO hero scenario.

## Candidate Criteria

- Publicly identifiable satellite name.
- TLE available from public sources.
- Story is easy for Indian judges to understand.
- Orbit makes visual demo interesting.

## Candidate Asset Types

- Earth observation satellites.
- Navigation satellites.
- Remote sensing satellites.
- Student/technology demonstration satellites if public TLEs are available.

## Scenario Requirement

The scenario may use real object identities with an injected, clearly labeled high-risk encounter if no live dramatic conjunction exists.
