# OrbitGuard Research Notes

This folder stores research that supports the science, positioning, and judge defense for OrbitGuard.

## Files

- `orbital-mechanics-notes.md`: propagation, conjunction, Pc, and maneuver references.
- `celestrak-data-notes.md`: public data source notes and snapshot strategy.
- `isro-assets-notes.md`: Protect ISRO watchlist research.
- `competitor-analysis.md`: why OrbitGuard is different from existing systems.

## Rule

Research notes should separate confirmed facts, assumptions, and demo simplifications.
