# Round 1 Build Order

## Goal

Ship a polished Protect ISRO vertical slice with enough architecture and tests to prove final-round seriousness.

## Order

1. FastAPI backend skeleton.
2. Catalog data service with deterministic fixtures.
3. Scenario engine with Protect ISRO.
4. Orbit propagation engine.
5. Conjunction screening engine.
6. Collision probability engine.
7. Avoidance maneuver planner.
8. Secondary risk screening.
9. AI briefing and report summary.
10. Mission console frontend.
11. Demo mode and offline replay.
12. Testing, deployment, and smoke checks.

## Round 1 Definition of Done

- Protect ISRO demo runs end-to-end.
- Before/after maneuver metrics are visible.
- Secondary screening result is visible.
- README, HLD, LLDs, methodology, and demo script exist.
- Submission checklist is complete.
